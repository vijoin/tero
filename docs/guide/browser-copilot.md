# Browser Copilot

![Browser Copilot](./img/browser-copilot.png)

Use your Tero agents right where you need them, on the page you’re working on. Copy information between the page and the copilot and avoid context switching.

## Setup

1. Install the extension from the [Chrome Web Store](https://chromewebstore.google.com/detail/tero-browser-copilot/fimbaimlephkekbpfdidmkenhjddimak).

:::: tip Pin it
After installing, pin the extension in your browser toolbar for quick access.
::::

2. Open any page and toggle the copilot:
   - Click the extension icon, or
   - Right‑click the page and choose “Browser Copilot”.
2. Click “Add Copilot” and enter your Tero instance URL, e.g., `https://trytero.abstracta.us`. Sign in if prompted.
3. You’ll see your set of Tero agents and can use them on any website.

Once open, it’s the same chat experience you already know. See [Use Agents](./chat.md) for tips, and explore new agents in [Discover Agents](./discover.md).
